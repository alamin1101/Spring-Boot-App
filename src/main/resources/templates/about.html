Micro library is a web based library management system. In this system, all control handled by an admin. When no admin are available then an admin can sign up with admin role automitacally. Then admin can login this system. Otherhand users can also sign up firstly with user role. Then he/she can login this system and just see all books by category, title or all books with available or non available.He/she also see his/her previous borrowed books with fine.When a user will come to the library to borrow books or return previous borrowed books with fine if needed. Admin will receive borrow books from users individually with fine if needed and also handle users new borrow book order.Admin also handle to add new books in the library and order new books.Admin see all books with category, title or all and also handle update or delete books.
